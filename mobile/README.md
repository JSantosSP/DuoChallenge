# Mobile App - DuoChallenge Game

## Instalación

```bash
npm install
```

## Configuración

Edita `.env` con la URL de tu API

## Ejecución

```bash
# Iniciar Expo
npm start

# Android
npm run android

# iOS
npm run ios
```

## Estructura

```
src/
├── components/   → Componentes reutilizables
├── screens/      → Pantallas principales
├── services/     → API calls
├── utils/        → Helpers
└── navigation/   → Configuración de navegación
```
